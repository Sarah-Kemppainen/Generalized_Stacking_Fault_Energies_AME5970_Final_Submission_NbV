## Each folder contaions

1. The `lmp_mcnpt.in` file

2. One `lmp_gsfe.in` file, where the integer number in line 54 is 10

3. The `data.CSRO` file generated by the CSRO simulation

4. The `log.lammps` file from the CSRO simulation

5. All 20 gsfe files, named `gsfe-1`, `gsfe-2`, etc, where 1 or 2 is the integer number in line 54 of the corresponding `lmp_gsfe.in` file

6. All 20 USFE values and the mean USFE value, provided in a text or excel file
